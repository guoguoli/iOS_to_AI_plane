# 2026-04-23 工作日志

## function_calling.py 修复

修复了两处 dashscope SDK 返回值访问方式的错误：

1. **AttributeError: 'dict' object has no attribute 'function'**
   - 原因：`tool_calls` 列表中每项是 `dict`，不是对象
   - 修复：`tool_call.function.name` → `tool_call['function']['name']`，`tool_call.id` → `tool_call['id']`

2. **KeyError: 'tool_calls'**
   - 原因：当模型不触发 function call 时，`message` 中没有 `tool_calls` 字段，直接 `.tool_calls` 抛 KeyError
   - 修复：改用 `response.output.choices[0].message.get('tool_calls')` 安全访问

3. **assistant 消息缺少 tool_calls 字段**
   - 修复：第二轮对话前，将 `tool_calls` 补入 assistant 消息体中

运行结果：3 个问题全部正常返回 AI 回复，EXIT:0。
