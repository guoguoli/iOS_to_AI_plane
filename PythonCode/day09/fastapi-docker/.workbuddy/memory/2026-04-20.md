# 2026-04-20 工作记录

## Docker 构建排错 - edu-ai-question-bank:v1.0

**问题1：DockerHub 拉取超时**
- 错误：`context deadline exceeded` 拉取 `python:3.11-slim`
- 原因：国内网络无法直连 DockerHub
- 修复：在 `~/.docker/daemon.json` 添加 `registry-mirrors`，配置三个国内加速器：
  - `https://docker.m.daocloud.io`
  - `https://dockerhub.icu`
  - `https://docker.nju.edu.cn`
- 重启 Docker Desktop 后生效

**问题2：Poetry 安装失败**
- 错误：`The current project could not be installed: No file/folder found for package edu-ai-question-bank`
- 原因：Poetry 2.x 默认尝试安装当前项目包，但 Dockerfile 中只复制了 `pyproject.toml`，没有 src 代码结构
- 修复：在 Dockerfile 的 `poetry install` 命令中添加 `--no-root` 参数

**最终结果：** 构建成功，镜像 `edu-ai-question-bank:v1.0` 大小约 480MB（131MB content size）
